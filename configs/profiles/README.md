# Config Profiles

Purpose: environment-specific config profile examples (dev, test, staging, prod).

Owner: SDKWork Knowledgebase maintainers.

Allowed content: safe checked-in profile templates and non-secret defaults.

Forbidden content: live credentials, secrets, local overrides, and runtime user config.

Related specs: `../../sdkwork-specs/CONFIG_SPEC.md`, `../../sdkwork-specs/ENVIRONMENT_SPEC.md`.
