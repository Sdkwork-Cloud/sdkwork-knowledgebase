# Config Defaults

Purpose: non-secret default config values for development and testing.

Owner: SDKWork Knowledgebase maintainers.

Allowed content: safe default values, feature flag defaults, and environment defaults.

Forbidden content: live credentials, secrets, production values, and local overrides.

Related specs: `../../sdkwork-specs/CONFIG_SPEC.md`, `../../sdkwork-specs/ENVIRONMENT_SPEC.md`.
